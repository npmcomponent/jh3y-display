How to create a display

Firstly, you need to work out what you want to display and how. This could be anything from pictures to information, even media.

Put your content into the correct stucture, basically divs with a container div.

Now theme it

Now you've got the content, you need to decide on how you're going to transition through it and how it should look.

You can see two example themes at :

  • https://github.com/jheytompkins/display-basic-top-to-bottom.theme
  • https://github.com/jheytompkins/display-basic-side-to-side.theme

those two example themes

These are themes based on 200px by 400px content. You can define whatever sizes you wish but I found this easier for those particular themes and with calculating the translate properties. Using a css preprocessor like less makes things a lot easier in this area because if I wanted to make a theme for a larger display, less can calculate the new translate properties from the new height and width I give it which saves me hassle when it comes to more complex themes.
